# 보고서 양식 모음

여기에 회사에서 실제 쓰는 양식과 제출 예시를 넣는다. Claude가 이 폴더를 읽고 보고서 스킬을 만든다.

## 넣을 것

- `주간보고/` — 양식 + 최근 제출본 2~3부
- `월간보고/` — 양식 + 제출본
- `기안서/` — 기안·품의 양식 + 예시
- `엑셀/` — 자주 하는 엑셀 작업의 "원본 → 완성본" 한 쌍

파일 형식은 hwp, hwpx, docx, xlsx, pdf 전부 가능. **실제 제출본이 많을수록 말투·표현이 정확해진다.**

## 스킬 제작 순서 (Claude 작업 지침)

1. 이 폴더의 양식·예시를 읽고 구조와 문체를 파악한다.
2. `.claude/skills/`에 스킬 생성: /주간보고, /월간보고, /기안서
3. 데이터 출처: 앱의 백업 JSON (메모 탭 → 백업 버튼) 또는 Supabase 직접 조회.
   - 주간보고 = 이번 주 완료 메모·조치 + 이번 주 점검(runs) + 특이사항(runs[ym].note)
   - 월간보고 = 이번 달 점검 이행률(완료/계획), 법정 필수(★) 이행, 조치·수리 내역, 월별 이력 특이사항
4. 출력: 양식과 같은 형식의 파일로 저장 (hwp 양식이면 docx로 생성 후 사용자가 붙여넣기).
5. 스킬은 저장소에 커밋해서 회사·집 공용으로.
